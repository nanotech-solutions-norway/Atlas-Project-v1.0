# Control Plane Overview

## Purpose

The Atlas control platform provides one operating surface for multiple GitHub-hosted websites.

## Core layers

1. Frontend control interface
2. Orchestrator API
3. Repository and connector adapters
4. Logging and archive layer

## Execution pattern

A user selects a website, scopes a task, reviews the AI-generated change plan, and approves draft or execution actions.

## Managed systems

- GitHub repositories
- Google Drive archive layer
- Future analytics and SEO sources

## Governance principle

Every AI-originated write should be reviewable, attributable, and reversible.
