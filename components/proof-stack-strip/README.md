# Proof Stack Strip

Purpose: keep trust assets near the decision point so Atlas feels credible and operationally grounded.

## Launch behavior
- Home: show 3–4 trust-oriented proof cards
- Solutions / Packages: proof adjacent to package claims
- Trust / Compliance page: deeper proof drawer or list

## Content rule
Proof cards must be tied to approved claims and should not imply secure delivery features that are not yet implemented.
