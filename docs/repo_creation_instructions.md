# Manual Repository Creation Instruction

Repository name: **Atlas Project v1.0**
Owner: **nanotech-solutions-norway**
Visibility: **private** (recommended)

After the repository exists:
1. Upload or push the contents of this pack
2. Set default branch to `main`
3. Create branch `develop`
4. Optionally create the tag `v1.0.0-governance-baseline`
5. Protect `main` if desired

This repository is intended to hold code-managed Atlas structure and governance artifacts, not to replace Wix page composition.
